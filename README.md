            <h2 id="problema">1. O Problema </h2>
                <p>Falta de ferramentas para gerenciar informações sobre seu pet, desde os cuidados 
                   mais básicos, como vacinação, alimentacão, cuidados de saúde, higiene e etc.</p>
            <h2 id="solucao">2. A Solução </h2>
                <p>
                    Criar um software que permita essa fácil interação entre o dono do pet, seus 
                    dados e necessidades, possibilitando assim maior facilidade na sua criação.
                </p>
